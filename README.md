# 어르신재가복지센터 웹사이트

GitHub Pages로 배포하는 재가복지센터 소개 웹사이트입니다.

## 배포 방법

1. 이 저장소를 GitHub에 업로드합니다.
2. GitHub 저장소 설정에서 Pages를 활성화합니다.
3. `data.json` 파일을 루트에 배치합니다.

## 관리자 사용법

### 로그인
- 우측 하단 "관리자 로그인" 버튼 클릭
- 아이디: `pcy6848`
- 비밀번호: `psj@30303`

### 공지사항/갤러리 수정
1. 관리자 모드로 로그인
2. 공지사항 또는 갤러리 추가/수정/삭제
3. "📥 JSON 다운로드" 버튼 클릭
4. 다운로드된 `data.json` 파일을 GitHub 저장소에 업로드/커밋
5. 모든 사용자에게 변경사항이 반영됩니다

### 이미지 추가 방법
1. 이미지 파일을 `assets/images/` 폴더에 업로드
2. GitHub Pages URL 형식으로 입력:
   ```
   https://사용자명.github.io/저장소명/assets/images/파일명.jpg
   ```
3. 또는 외부 이미지 호스팅 서비스 URL 사용

## 파일 구조

```
├── test.html          # 메인 HTML 파일
├── data.json          # 공지사항 및 갤러리 데이터
├── assets/
│   ├── images/       # 갤러리 이미지 파일들
│   └── background.png # 배경 이미지
└── README.md         # 이 파일
```

## 주의사항

- 이미지는 반드시 URL 형식으로 입력해야 합니다.
- base64 이미지는 사용할 수 없습니다 (GitHub Pages 제한).
- 변경사항을 반영하려면 `data.json` 파일을 GitHub에 커밋해야 합니다.
