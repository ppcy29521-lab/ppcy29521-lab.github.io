# 네이버 검색 최적화 가이드

## ✅ 완료된 작업

1. **메타 태그 추가**
   - 사이트 설명 (description)
   - 키워드 (keywords)
   - Open Graph 태그 (og:title, og:description, og:image)
   - 구조화된 데이터 (JSON-LD)

2. **SEO 파일 생성**
   - `robots.txt` - 검색엔진 크롤러 안내
   - `sitemap.xml` - 사이트 구조 안내

## 📋 네이버 검색 등록 방법

### 1단계: 네이버 서치어드바이저 가입

1. **네이버 서치어드바이저 접속**
   - https://searchadvisor.naver.com/ 접속
   - 네이버 계정으로 로그인

2. **사이트 등록**
   - "웹마스터 도구" → "사이트 추가"
   - 사이트 URL 입력: `https://ppcy29521-lab.github.io`
   - 사이트 이름: `어르신재가복지센터`

### 2단계: 사이트 소유 확인

**방법 1: HTML 파일 업로드 (권장)**
1. 서치어드바이저에서 "HTML 파일 업로드" 선택
2. 제공된 HTML 파일을 다운로드
3. GitHub 저장소 루트에 업로드
4. "확인" 버튼 클릭

**방법 2: 메타 태그 추가**
1. 서치어드바이저에서 "메타 태그" 선택
2. 제공된 메타 태그 코드 복사
3. `index.html`의 `<head>` 섹션에 추가
4. GitHub에 업로드 후 "확인" 버튼 클릭

### 3단계: 사이트맵 제출

1. 서치어드바이저 대시보드에서 "요청" → "사이트맵 제출"
2. 사이트맵 URL 입력: `https://ppcy29521-lab.github.io/sitemap.xml`
3. "제출" 클릭

### 4단계: 검색 최적화 설정

1. **사이트 정보 수정**
   - 사이트 설명: "익산시 어르신재가복지센터는 전북특별자치도 익산시에 위치한 전문 재가복지센터입니다. 방문요양, 목욕지원, 식사지원 등 어르신을 위한 따뜻한 돌봄 서비스를 제공합니다."
   - Open Graph 제목: "익산시 어르신재가복지센터 | 방문요양 서비스 | 전북 익산 재가복지"
   - Open Graph 설명: "익산시 어르신재가복지센터는 전북특별자치도 익산시에 위치한 전문 재가복지센터입니다. 방문요양, 목욕지원, 식사지원 등 어르신을 위한 따뜻한 돌봄 서비스를 제공합니다. 연락처: 063-841-8880"

2. **키워드 설정**
   - 주요 키워드: "익산시 재가복지센터", "어르신재가복지센터", "익산 재가복지", "익산시 복지센터", "전북 익산 재가복지", "방문요양 익산"

### 5단계: URL 제출

1. 서치어드바이저에서 "요청" → "URL 제출"
2. 메인 페이지 URL 제출: `https://ppcy29521-lab.github.io/`

## ⚠️ 주의사항

1. **검색 등록 후 시간 필요**
   - 사이트 등록 후 검색 결과에 반영되는데 **1-2주** 정도 소요될 수 있습니다
   - 첫 등록은 더 오래 걸릴 수 있습니다

2. **정기적인 업데이트**
   - 콘텐츠를 정기적으로 업데이트하면 검색 순위가 유지됩니다
   - 공지사항, 갤러리를 자주 업데이트하세요

3. **키워드 과다 사용 금지**
   - 키워드를 자연스럽게 사용하세요
   - 과도한 키워드 반복은 오히려 불이익을 받을 수 있습니다
